Related Software/Resources

Remailer Clients

The Mutt E-Mail Client has an interface to call the Mixmaster binary. For Emacs Erik wrote an interface: mixmaster.el.

The JBN project on sourceforge was just started in order to maintain JBN again and bring it up to date. (old versions)

Mixlet is a Mix Client in Java. It does not rely on Mixmaster itself to build the messages but rather implements the required procedures itself.

Deacon is developing a Nym-Proxy which lets you use your usual Mail and News clients while still using a Nym. This is for Unixes.

QuickSilver is an actively maintained and developed client for Win32. It doesn't call the Mixmaster binary but instead links against the mix.dll build from Mixmaster's sources.

Section 6 of the alt.privacy.anon-server FAQ perhaps has a list of more remailer clients.

Other Remailers

Reliable is a popular piece of remailer software for Microsoft Windows. It is type I (cypherpunk) only but has type II callbacks. So if you additionally install Mix2 or Mix3 on your system it can also do type II.

Deacon also created the Reliable2 project here on SourceForge in order to bring it up to current standards.

Mixminion will become the next generation remailing network (Type III). It's not yet ready to provide you with anonymity but it's well on its way.

Free Haven's Anonymity Bibliography has pointers to all the important papers in the anonymity field. It includes recommended reading lists to get new developers up to speed on remailer designs and related systems.

Nymservers

The source for the original newnym style nymserver that runs nym.alias.net can be dowloaded via finger from finger:source@nym.alias.net (you can also send an email to source@nym.alias.net).

There also is a project that makes the nym.alias.net source work with newer PGP and GnuPG versions. The project web site is at http://sourceforge.net/projects/nymserv.

Pingers

For remailers to work in a reliable way it is imperative that remailers and users have the latest keys and reliability statistics of other remailers. Such data is collected by programs called Pingers. Either you operate your own or fetch the data from one of the many existing pingers.
