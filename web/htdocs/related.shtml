Other Remailers

Reliable

Reliable is a very popular remailer software for Microsoft Windows. It is type I (cypherpunk) only but has type II callbacks. So if you additionally install Mix2 or Mix3 on your system it can also do type II.

Pingers

For remailers to work in a reliable way it is imperative that remailers and users have the latest keys and reliability statistics of other remailers. Such data is collected by programs called Pingers. Either you operate your own or fetch the data from one of the many existing pingers.

Sorted alphabetically:

  • echolot by Peter Palfrader
  • pingstats by cmeclax
  • remlist by Chritian Mock
